🚖 City Ride Analysis Dashboard

 Project Overview
This project presents an interactive **City Ride Analysis Dashboard** built using Power BI. It provides insights into ride bookings, revenue trends, vehicle performance, and customer behavior to support data-driven decision-making.

 Objectives
* Analyze total bookings and successful rides
* Track revenue generation and booking value
* Understand vehicle-wise performance
* Analyze ride distance and revenue per km
* Identify cancellation patterns and booking status

 🛠️ Tools & Technologies
* Power BI
* Microsoft Excel
* Data Visualization Techniques

 📊 Dashboard Features
 🔹 Key Metrics (KPIs)
* 🚖 Total Bookings: 150K
* ❌ Total Cancelled: 38K
* 💰 Total Booking Value: 13M+
* 📊 Successful Booking Value: 12M+
* 📏 Average Distance Travelled: 26 km
* 🌍 Total Distance Travelled: 600K+ km
🔹 Visual Insights

* 📈 Bookings Trend by Month – Identifies seasonal ride demand
* 📊 Revenue by Payment Method** – UPI, Cash, Card analysis
* 🚗 Vehicle Type Analysis – Auto, Bike, eBike, Go Mini, Sedan, etc.
* 🧭 Revenue per KM by Vehicle Type– Performance comparison
* 📅 Ride Distance Trends (Daily/Monthly)
* 📉 Booking Status Analysis – Completed, Cancelled, No Driver, etc.

🔹 Filters Available
* Date Range
* Vehicle Type
* Payment Method
* Day Part (Morning, Evening, Night)

## 💡 Key Insights

* Ride bookings show fluctuations across months indicating demand patterns
* UPI and digital payments dominate transactions
* Certain vehicle types generate higher revenue per km
* Cancellation rate highlights operational inefficiencies
* Distance trends help understand ride distribution and usage
